# Z shell

* [Oh My ZSH!](https://ohmyz.sh/)
* [GNU Make](https://www.gnu.org/software/make/)

## Getting started

``ln -s $PWD/.zshrc $HOME/.zshrc``

or try it out if you have just cloned it (to your home directory) and uptake new changes:

```shell
source ~/.zshrc
```
